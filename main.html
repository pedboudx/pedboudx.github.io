Shopping Cart Console Application

C++

Console Application built using C++ made for a class project, it aims to emulate the functionality of an online shopping cart using concepts learned in my Object Oriented Programming in C++ class. The program begins with a welcome message, prompting the user to proceed to the list of available items or close the application. If the user chooses to proceed, they are presented with the available items and the number of items in their cart. The user can select an item by entering a corresponding number and view additional information, then they can add the item to their cart or go back. Once the user is finished adding items, they can proceed to view their cart, displaying all items they have added and then prints a receipt in a txt file.

Password Manager with Encryption

C++

Console Application built using C++, stores and reads user login information from an encrypted file. Upon entering the program, makes the user set a master key which is used every time they open the program in the future. Then presents the user with the options to manage their currently stored login informations, look up a specific website's login information or view the entire list of saved logins. Once the user exits the program, it ensures that all information is encrypted again.
